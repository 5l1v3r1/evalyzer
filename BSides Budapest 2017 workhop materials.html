windbg comes from Windows SDK - only install Debugging tools
    win7: https://www.microsoft.com/en-us/download/details.aspx?id=3138
    win8: https://developer.microsoft.com/en-us/windows/downloads/windows-8-1-sdk
    win10: https://developer.microsoft.com/en-US/windows/downloads/windows-10-sdk

IE debug symbols would be downloaded directly from MS symbol servers
windbg will be installed into: 
C:\Program Files (x86)\Windows Kits\10\Debuggers\x86
WinDBG ultraprimer: here WinDBG cheat sheet: http://windbg.info/doc/1-common-cmds.html
